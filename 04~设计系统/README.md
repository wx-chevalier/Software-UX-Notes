1. 原子设计原则（Atomic Design）
   核心思想：将界面拆解为原子（基础组件，如按钮）、分子（组合组件，如搜索栏）、 organism（模板，如首页布局）、页面（具体页面实例），通过模块化构建一致性体验。
   应用场景：
   大型团队协作设计时，统一组件库和设计规范。
   快速迭代时复用组件，减少重复劳动。

2. 8pt 网格系统（8pt Grid System）
   核心思想：以 8 为基数进行界面元素的间距、尺寸设计，确保视觉一致性和适配性。
   应用场景：
   移动端界面中，元素间距常用 8、16、24pt 等倍数。
   响应式设计中，不同屏幕尺寸按 8 的倍数缩放。
